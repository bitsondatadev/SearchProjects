href="/report/">Corrections &amp; Updates</a> </li> </ul> </div> <div id="feature"> <span class="feature_title">Anonymous FTP Use</span> <p> If you have an anonymous FTP directory, it will be created as <code>/p/ftp/pub/<i>login</i></code> and owned by you. This directory is deleted when your CS department account is deleted.</p> <p> Please be conservative in your space use and only put out there what must be there.</p> <p> Edit <code>/p/ftp/pub/<i>login</i>/.message</code> as appropriate. This message gets displayed when someone cd's into the directory (once per ftp session).</p> <p> Do not create any world writable directories (if you think you need these, talk to <a href="mailto:software@cs.purdue.edu">Facilities Staff</a>. We reserve the right to delete the <code>/p/ftp/pub/<i>login</i></code> directory completely if we find them.</p> <p> Be careful not to put any copyrighted or licensed material out there unless you have explicit permission from the appropriate people/organization. If you have any questions about what is OK, please ask <a href="mailto:software@cs.purdue.edu">Facilities Staff</a>.</p> <p> The location for your directory should be advertised as <code>/pub/<i>login</i></code> on ftp.cs.purdue.edu, or the URL ftp://ftp.cs.purdue.edu/pub/<i>login</i></p> <div style="height: 15px; width: 10px;"></div> <p> Contact Us: <a href="mailto:software@cs.purdue.edu">Facilities Software Staff</a><br /> <span style="color: #777">Last Updated: July 28, 2006 12:54PM</span > </p> <div style="height: 15px; width: 10px;"></div> </div> <div id="info"> <div> <span class="header">Purdue University Policies</span> <a href="http://www.purdue.edu/policies/">Purdue Policies Homepage</a> <a href="http://www.purdue.edu/policies/records/viiia4.html">Purdue FERPA Policy</a> </div> <div> <span class="header">Purdue IT Policies</span> <a href="http://www.purdue.edu/policies/information-technology.html">IT Policies Home</a> <a href="http://www.purdue.edu/policies/information-technology/viib1.html">Authentication and Authorization (VII.B.1)</a> <a href="http://www.purdue.edu/policies/information-technology/viib2.html">Privacy for Electronic Information (VII.B.2)</a> <a href="http://www.purdue.edu/policies/information-technology/c-34.html">Data Security and Access Policy (C-34)</a> <a href="http://www.purdue.edu/policies/information-technology/c-42.html">Purdue Pages on the World Wide Web, Policy for (C-42)</a> <a href="http://www.purdue.edu/policies/information-technology/viia1.html">Electronic Mail (VII.A.1)</a> </div> </div> </div> <div style="clear: both;" /> <div id="footer"> <div style="font-size: 80%;font-weight: 800;float: left;"> Computer Sciences, 305 N.
