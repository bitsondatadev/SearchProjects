<HTML> <HEAD> <TITLE>CS 175 Lab</TITLE> </HEAD> <BODY BGCOLOR="WHITE"> <FONT COLOR="#CE9C33"><H2>CS 175 Lab</H2></FONT> The CS175 Lab consists of 19 Intel machines running Windows NT 4.0. This computer laboratory is used by graduate and undergraduate groups for a variety of class projects, including computer graphics, networking, user interface design, and operating systems projects. Its use is reserved for students enrolled in classes scheduled for using the lab.
