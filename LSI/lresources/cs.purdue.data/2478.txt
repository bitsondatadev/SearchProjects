<html> <title>CS490T - VNCR</title> <h1>VNCR </h1> <center><a href="VNCR.ppt">End of Semester Powerpoint Presentation</a></center> <hr> <center>You should check out our <a href="http://vncr.sourceforge.net/">Project Page</a> or our <a href="http://vncr.sf.net/wiki/">wiki</a>.</center> <p>The goal of this project is to create a VNC Recorder - a program, based on VNC code, that will record what is seen on the display of a computer for later playback. Audio will also be recorded.</p> <p>Possible uses include recording of presentations, software tutorials, and source code explanations.</p> <p>This will initially be implemented under windows using C#.net, using the <a href="http://www.realvnc.com/docs/rfbproto.pdf">RFB protocol standards</a> and leveraging <a href="http://lame.sourceforge.net/">LAME</a> for audio.</p> <p>Cross-platform support will eventually be extended using the <a href="http://www.mono-project.com">Mono project</a>.</p> <p>Goals:</p> <ul> <li>Custom file format including recorded video and sound <li>Recorder with VCR-like controls <li>Player with VCR-like controls <li>Editor (cutting/pasting/splicing, insertion of frames, fading?) <li>Browser player plugins (IE, Firefox) </ul> </html> </html>
