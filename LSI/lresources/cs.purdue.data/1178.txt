<HTML> <TITLE>Arachne Threads Homepage</TITLE> <IMG ALT="Arachne" ALIGN=MIDDLE SRC="pics/arachne_trans.gif"> <P> <BR> <DT><H2><IMG ALT="o " SRC="pics/bluedot.gif"> Description</H2> <DD><I>Arachne Threads</I> is a user-level threads package capable of dynamically migrating threads during execution across heterogeneous platforms. Arachne accomplishes this task by inserting code to handle translation tasks via a preprocessor.
