<html> <head> <title>The Tonitza Project</title> </head> <body> <h1>The Tonitza Project</h1> Principal Investigator: <a href="../faculty/dcm.html">Dan C. Marinescu</a><p> Sponsor: NSF <P> <img src="goldRule.gif" width=100%><p> Tonitza is an interactive graphics package for data visualization, analysis, and manipulation. It is aimed at the field of structural biology, but it can be used to visualize any vector field data. Tonitza consists of: (a) a Graphical User Interface (GUI), (b) an I/O module responsible for reading/writing files, for input data format recognition, for conversion between data formats, and for handling I/O errors, (c) a computation module which provides capabilities for data rotation, computing the correlation of two data sets, composite map generation, and obtaining statistical information, and (d) a visualization module, which allows the user to display his/her data in various two- and three-dimensional representations, to rotate, translate, and scale the objects. <P> For more information please see <a href="/homes/sb/Projects/Tonitza/tonitza.html">Tonitza</a><p> <img src="goldRule.gif" width=100%> </body> </html> </html>
