<!-- $Id: pdevpe.html,v 1.3 1994/02/28 19:41:15 acc Exp $ --> <HEAD> <TITLE>PDEVPE</TITLE> </HEAD> <PRE><H1><IMG ALIGN=MIDDLE SRC="../../gifs/pdelab/pdevpe-small.gif"></a> A Virtual Parallel Environment </PRE> </H1> <P> <b>PDEVpe</b> is a framework within the PDELab environment which provides a platform for developing portable parallel PDE solvers, along with facilities and tools for analyzing their performance. <P> <HR> PDEVpe is a interactive environment which provides the following virtual parallel tools and facilities: <MENU> <LI>Kernals (BLAS, linear systems solvers, etc) for developing parallel components of the numerical simulation process. <LI>Communication and scheduling libraries (MPI,PICL, PARMACS, PVM) for making non-portable code portable. <LI>Parallel languages (HPF). <LI>Parallelizing methodologies for re-using sequential code in a parallel environment (REUSE). <LI>Instrumentation tools for modifying software to collect data necessary for performance analysis, and visualization tools for viewing performance data related to parallel processing. <LI>Skeletons for parallel methods (PTS) <LI>Matrix decomposition and geometry splitting techniques for defining the top-level parallelism in the PDE computation. </MENU> <P> <BODY> </BODY> <P> <HR> <ADDRESS><IMG SRC="../../gifs/pdelab/pdelab-logo-emboss.gif"> For further information contact pdelab@cs.purdue.edu</ADDRESS> pdelab@cs.purdue.edu</ADDRESS>
