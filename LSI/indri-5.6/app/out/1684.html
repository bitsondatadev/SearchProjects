 What is SpeechP3?

SpeechP3 was designed to make a media player that did not require you to be in front of your computer at all times.  Instead of the standard method of pointing and clicking, operations would be carried out through voice recognition.  For instance, if you wanted to start playing your media, all you have to do is say "Computer Play" and it would start playing from the current spot in your play-list.  It also supports other commands as will be described later.  If you would like to view a more detailed description of what SpeechP3 can do, feel free to take a look at our final presentation. It should provide plenty of information.  You can access it here.  Also, if you would like to see our users manual, you can access it here.

 

Why SpeechP3?

So the obvious next question is why do we need this?  How lazy are we that we can't lift up a finger and click on then next track button when a song comes on that you don't like?  First of all, the answer is very lazy.  But this is not the only reason SpeechP3 was created.  It was mainly created for situations where your hands would be occupied doing something else, and the computer might even be all the way across the room.  So it might become handy, for instance, if you're cooking or cleaning and want to listen to music at the same time.  Its quite a nuisance to go all the way over to your computer when a bad song comes on, and also your hands might be saturated in cooking supplies.  Another good use for this would be if you're exercising, and you are using both your hands, but all the sudden Chumbawumba's "Tub-Thumping" comes on your player.  On one hand , you don't want to get up and stop exercising, and on the other, your hands will be too tightly clamped over your ears to click the next button anyways.  So all you have to do is say "Computer, Next" and you'll be in the clear again.  And one other very important use for our program would be to install it into cars.  Accidents are caused every year by drivers who do too many things at the same time, and our application would help reduce this multi tasking by taking care of media controls.

SpeechP3 Project

 

Download the zip file above and extract it using your favorite (or least favorite if you are so inclined) zip program. Wherever you extract it, there should be a setup sub-directory. To install SpeechP3, you should go into said directory and run the setup.exe program. It will then ask you where you would enjoy SpeechP3 to be located on your computer, and then install it there. The setup program will also add a link to your Start menu and your Desktop.

 

The source code is also available from the aforementioned zip file. Instead of the setup sub-directory, try looking in the source one. From there, you can run the Visual Studio Solution file, or look at the C# code directly.
