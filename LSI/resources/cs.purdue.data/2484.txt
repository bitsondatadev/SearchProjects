Whether you are a freshman that is new to campus or a super senior who knows Purdue's like the back of your hand this application will make your life much easier.</p> <p style="font-size:18px"> <b> Features Include: </b> <dl> <dt> <b>Lab Check</b> (<i>Only works during hours of operation</i>)</dt> <dd>- Checks to see which Computer labs are available on campus and how many computers are being use in that lab</dd> <dd>- Allows the user to check for Windows or Mac Machines depending on preference</dd> <dd>- Shows the user where the selected lab is located on Google Maps</dd> <br /> <dt><b>Dining Court Menu</b></dt> <dd>- Displays the dining court menu for the dining court of choice</dd> <br /> <dt><b>News Feed</b></dt> <dd>- Shows Purdue news for selected topic of interest</dd> <br /> <dt><b>Book Search</b></dt> <dd>- Allows the user to search for books in Purdue's library catalog </dd> <dd>- Displays all library information including the availability of books </dd> <dl> </p> </div> <h2> Downloads </h2> <p> To use this application download one of the apk files below </p> <h2> Compile and use </h2> <p> To use the source file, simply go to eclipse and select import prject, select the downloaded zip file.
