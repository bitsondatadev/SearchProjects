/> <div> <p> Dr. Hosking's research lies at the intersection between programming languages and database systems, focusing on the integration of language and database functionality for efficient data management. Particular topics of interest include interpretation, compilation, and optimization of object-oriented persistent/database programming languages, and empirical performance evaluation of experimental prototype systems. His current research explores language and compiler support for run-time object management (e.g., garbage collection, persistence, resilience, distribution and security) in the context of the Smalltalk, Modula-3, and Java programming languages. </p> </div> <div> <strong>Selected Publications</strong> <div style="margin: 0 0 10px 0;"> Jones R, Hosking A, Moss E, "The Garbage Collection Handbook: The Art of Automatic Memory Management", Chapman and Hall, 2011. </div> <div style="margin: 10px 0 10px 0;"> Kalibera T, Pizlo F, Hosking AL, Vitek J, "Scheduling real-time garbage collection on uniprocessors", <em>ACM Transactions on Computer Systems 3</em>(1), pages 8:1-29, August 2011. </div> <div style="margin: 10px 0 10px 0;"> McGachey P, Hosking AL, Moss JEB, "Class transformations for transparent distribution of Java applications", <em>Journal of Object Technology 10</em>, pages 9:1-35, August 2011. </div> </div> <div> <strong>Research Funding</strong> <div style="margin: 0 0 10px 0;"> Jan Vitek and Tony Hosking, <em>CPA-CPL Certified Garbage Collection for Highly Responsive Systems</em>, National Science Foundation, 8/1/2008-7/31/2012. </div> <div style="margin: 0 0 10px 0;"> Tony Hosking, <em>ECOOP 2012 CONFERENCE</em>, Assoc Internationale pour les Tech Objet, 4/1/2011. </div> <div style="margin: 0 0 10px 0;"> Tony Hosking, <em>Ibm Eclipse Innovation Award</em>, IBM, 1/1/2003. </div> <div style="margin: 0 0 10px 0;"> Tony Hosking, <em>Microsoft: Language &amp;
