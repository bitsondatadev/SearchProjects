FACE="Arial,Helvetica"> List of Topics (By Week):<P> 1. Introduction to computers and software: Computer basics, software design, coding, testing, primitive types and expressions, string class, scanner class, printf-style format strings, typesafe enums<P> 2. Flow of Control: Branching (single and multiple), loops (fixed and variable repetition), booleans<P> 3. Classes and methods: Object-based programming, class and method definitions, information hiding, encapsulation, objects, references<P> 4. More about classes and methods: Accessor and mutator methods, constructors, overloading, variable number of arguments, static methods, static variables<P> 5. Arrays: Array basics, arrays in classes and methods, programming with arrays and classes, sorting arrays, searching arrays, multidimensional arrays<P> 6. Inheritance: Object-oriented programming, inheritance basics, programming with inheritance, overriding, dynamic binding, polymorphism, interfaces<P> 7. Exception handling: Basic exception handling, using exception classes, defining exception classes<P> 8. Recursion: Basics of recursion, programming with recursion<P> 9. Abstract data types: Stacks, queues, linked lists, trees<P> 10. Collections framework: Vector, array, list, set, map, queue, generic types, generalized iterator for loop, autoboxing and unboxing<P> 11. Streams and file I/O: Overview of streams and file I/O, text file I/O, binary file I/O, object I/O with object streams<P> 12. Reuseable software: Libraries, packages, more on inheritance, dynamic binding, polymorphism<P> 13. Event-driven programming and special categories of classes: Graphical user interfaces (GUIs), action events, action listeners, container classes, applet classes, inner classes<P> 14. Advanced topics: Threads, mutual exclusion, deadlock, livelock, remote method invocation (RMI)<P> 15. Advanced topics: Design patterns, software engineering, testing, features of large projects<P> 2005.08 </FONT> </TABLE> </BODY> </HTML> </HTML>
